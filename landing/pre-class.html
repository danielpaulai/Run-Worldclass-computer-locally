1. Install Ollama

Ollama is the runtime that runs the AI model on your laptop. Free, open source. Easiest way, works on every Mac:

Download Ollama for Mac Drag to Applications, launch once. No Homebrew required.

Already have Homebrew? You can use Terminal instead:

brew install ollama

If you see command not found: brew, that means Homebrew isn't installed. Use the download button above, you don't need Homebrew.

2. Start Ollama

If you used the download above: Ollama is already running. Look for a llama icon in your menu bar (top right of your screen). That means it's live.

If you used Homebrew instead: Start Ollama as a background service once, it'll auto-start every time you log in:

brew services start ollama

Verify either way worked: curl http://localhost:11434/api/version should print JSON with a version number.

3. Pull the model

This downloads a 4.7 GB file. Do it on WiFi, not cellular.

ollama pull qwen2.5:7b

Takes 2 to 10 minutes depending on your internet speed.

If something breaks

"Ollama not reachable" in the app

Ollama isn't running. Start it:

Mac:

brew services restart ollama

Linux:

sudo systemctl restart ollama

Windows: quit Ollama from the system tray, then relaunch from Start menu. Or run ollama serve in PowerShell.

"No models installed" in the app

The model didn't download. Run the pull again:

ollama pull qwen2.5:7b

If it fails partway, just run it again. It resumes automatically.

Mac: "Purely Personal can't be opened because it is from an unidentified developer"

Right-click the app in Applications, choose Open, then Open again in the dialog. One time only.

If the right-click menu doesn't show Open: go to System Settings → Privacy & Security, scroll down, click Open Anyway next to the Purely Personal line.

Windows: "Windows protected your PC"

Click More info, then Run anyway.

My laptop has 8 GB RAM. Will it work?

Yes. qwen2.5:7b needs about 5 GB. It'll be slower than on a 16 GB machine but it will work. If it feels too slow, you can switch to a smaller model during class: ollama pull qwen2.5:3b.
