## 1. The 4-Column Worksheet

**Source**: The Program Promise™, The Risk Reversers™, The Bonus Builder™, The Scarcity Engine™, The Pricing Hacks™
**Use**: When teaching a framework with best practices, examples, and participant fill-in

### Structure
```
[Header: The [Name]™]
┌──────────────┬──────────────┬──────────────┬──────────────┐
│  Column A    │  Column B    │  Column C    │  Column D    │
│  (header bg) │  (header bg) │  (header bg) │  (header bg) │
├──────────────┼──────────────┼──────────────┼──────────────┤
│ Pre-filled   │ Pre-filled   │              │              │
│ bullet list  │ bullet list  │  "Current"   │   "Ideas"    │
│ (reference)  │ (examples)   │  (blank)     │   (blank)    │
│              │              │              │              │
│              │              │ ┌──────────┐ │              │
│              │              │ │ Insight  │ │              │
│              │              │ │ (blank)  │ │              │
│              │              │ └──────────┘ │              │
└──────────────┴──────────────┴──────────────┴──────────────┘
[Footer: Brand | The [Module] • [Framework Name™]]
```

### Column Configurations

| Worksheet | Col A | Col B | Col C | Col D |
|-----------|-------|-------|-------|-------|
| Program Promise™ | Goal | Examples | Current ("It's designed to...") | Ideas |
| Risk Reversers™ | Best Practice | Examples | Current | Ideas |
| Bonus Builder™ | Best Practice | Examples | Current | Ideas |
| Scarcity Engine™ | Best Practice | Examples | Current | Ideas |
| Pricing Hacks™ | Best Practice | Examples | Current | Ideas |

### Key Rules
- Columns A and B are PRE-FILLED with reference content (bullet list, 11px, #3d3d3d)
- Columns C and D are BLANK (white fill, for participant writing)
- Column C often has a sub-section labeled "Insight" at the bottom
- Column widths: A=25%, B=25%, C=25%, D=25% (equal)
- Minimum cell height: 300px (give real writing room)
- Header row: #e2dfd9 background, 12px bold text

### Pre-fill Content by Type

**Program Promise™**
- Goal: Specific result, Set timeframe, Small promises, Aspirational
- Examples: Thin thighs in 30 days, CLIENTS, 20-day New Patient Bootcamp, Million Dollar Coach, $10K in 10 weeks
- Current prompt: "It's designed to..."

**Risk Reversers™**
- Best Practice: Reward good behaviour, Action based, No questions asked, Make it safe to start
- Examples: Rob Nixon: $300K in extra profit every year provided you X Y Z, I'll buy it back from you at full price and give you $10 for a plate of tacos, 120 day "one full cycle" Love It Or Leave It

**Bonus Builder™**
- Best Practice: Congruence, Accelerators objection overcomers, Sales and intimacy opportunities, Reward fast action
- Examples: 1:1 small group call, Done for you: Sidekick, Tools: CMM website, Scripts and templates: webinar template

**Scarcity Engine™**
- Best Practice: Lower supply to increase demand, Chunk down the milestones, Fast action bonuses, Sell past the close, No fake scarcity
- Examples: 10 spots in CMM Pro, The first 5 get..., We start on Tuesday, Our next intake is August 4

**Pricing Hacks™**
- Best Practice: ROI, Pay plan language, Align with outcomes, 3's and 5's
- Examples: $600/week, $20K PIF or $2K/month, $1,500 $2,000 $2,400

## 2. The Framework Diagram

**Source**: The Offer Diamond™, The Risk:Reward Ratio™, Market-Program-Offer triangle
**Use**: The central visual model that IS the teaching

### Diamond Shape (5 facets)
```
         ┌───────────────────────┐
        ╱ Pay Plan    Bonuses     ╲
       ╱                           ╲
      │  Guarantee      Promise     │
       ╲                           ╱
        ╲     Urgency/Scarcity    ╱
         └───────────────────────┘
```
- Pentagon/diamond shape: stroke #3d3d3d, fill none, strokeWidth 2
- Labels placed OUTSIDE the shape with small connecting lines
- Each label has a small medal/ribbon icon (optional) or traffic light dot
- Viewport: centered in 960×680 container

### Seesaw / Balance (Risk:Reward)
```
    REWARD ──────────── RISK
         ╲            ╱
          ╲──────────╱
               ╱╲
              ╱  ╲
             ╱    ╲
```
- Horizontal beam: stroke #3d3d3d, strokeWidth 2
- Triangle fulcrum: stroke #3d3d3d, fill none
- Left side label: "REWARD" (items below: Promise, Bonuses)
- Right side label: "RISK" (items below: Guarantee, Payment Plan)
- Bottom center: "Urgency & Scarcity" (the fulcrum weight)

### Triangle / Pyramid
```
          ╱╲ Pipeline
         ╱──╲ Traffic
        ╱────╲ Conversion
       ╱──────╲ Program | Offer
      ╱────────╲ Market
```
- Isoceles triangle: stroke #3d3d3d, fill none, strokeWidth 2
- Horizontal dividers inside at each level
- Labels inside each section, bold italic, white on dark bg version OR dark text on light bg version
- Dark version: #1a1a1a background, text in white/gray
- Light version: #f0eeea background, text in #3d3d3d

## 3. The 3-Column Grid (Whiteboard Style)

**Source**: Fears/Mistakes/Tips butcher paper, Wins/Lessons/Upgrade review
**Use**: Quick categorization, group brainstorming, reflection

### Structure
```
┌──────────────┬──────────────┬──────────────┐
│   Column A   │   Column B   │   Column C   │
│   (header)   │   (header)   │   (header)   │
├──────────────┼──────────────┼──────────────┤
│ • __________ │ • __________ │ • __________ │
│ • __________ │ • __________ │ • __________ │
│ • __________ │ • __________ │ • __________ │
│ • __________ │ • __________ │ • __________ │
│ • __________ │ • __________ │ • __________ │
│ • __________ │ • __________ │ • __________ │
└──────────────┴──────────────┴──────────────┘
```

### Common Configurations
- **Fears / Mistakes / Tips** — for reviewing a framework after teaching
- **Wins / Lessons / Upgrade** — for weekly review
- **Insights / Progress / Stuck** — for control room (with Action sub-box in Stuck column)
- **Problem / Fix / Next Step** — for tactical fix worksheets
- **Before / During / After** — for process mapping

### Key Rules
- Headers: blue marker effect (bold, slightly larger) or #e2dfd9 bg
- Bullet lines: horizontal rules with bullet dots, 6-8 per column
- Each bullet line: 1px solid #d0cdc8, with small dot at left
- Minimum height: 400px (fill-in space)

**Use**: Multi-week program overview with connected milestones

```
│Kickoff │ Week 1 │ Week 2 │ Week 3 │ Week 4 │ Week 5 │

├────────┼────────┼────────┼────────┼────────┼────────┤

│   ─────┼──┌──┐──◆──┌──┐──◆──┌──┐──◆──┌──┐──◆──┌──┐──→
│        │  │  │  │  │  │  │  │  │  │  │  │  │  │  │  │
│        │  └──┘  │  └──┘  │  └──┘  │  └──┘  │  └──┘  │

│        │  └──┘  │        │        │  └──┘  │        │
└────────┴────────┴────────┴────────┴────────┴────────┘
```

### Key Rules
- 6-7 columns (Kickoff + weeks)
- Central horizontal line with diamond connectors (◆) at each week transition
- Boxes above and below the line = deliverables/milestones for that week
- Kickoff = circle (○) at the start
- Arrow (→) at the end showing continuation
- All shapes: stroke #3d3d3d, fill #fff, border 1px solid #d0cdc8
- Column headers: bold + topic name

## Generating Complete Sets

When a user requests a "full set" for a session, generate in this order:

1. **Hero slide** (dark bg, session title)
2. **Today's Playlist page** (showing all video/training topics for the session)
3. **Framework diagram** (the central model)
4. **Worksheets** (one per sub-topic, usually 3-5)
5. **Workbench overview** (thumbnails of all worksheets)
6. **"Next..." action page**
7. **Dot grid notes page**
8. **"Coming Up Next" preview page** (teaser for next session)

This matches the exact page order of the Taki Moore Launchpad workbook.
