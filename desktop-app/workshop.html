Three things to try this week.

Pick a file or folder with the button, then copy the generated command into Terminal and press Enter. One per day this week is a habit by Friday.

1

Summarize a confidential doc

A contract, financial report, meeting notes \u2014 anything you'd never paste into ChatGPT.

./ai summarize tldr <your file>

Paste this into Terminal and press Enter.

2

Index a folder and ask it questions

Point the tool at a folder of your own work \u2014 notes, past emails, project docs. It becomes a searchable memory.

./ai local-rag ingest <your folder>

After ingest finishes, ask questions with ./ai local-rag query "your question".

3

Run the privacy audit

Proves to you (and your boss) that nothing is leaving your machine. No file needed.

./ai privacy-audit check

Then try it yourself \u2014 turn off WiFi, run a chat, watch it still work.
