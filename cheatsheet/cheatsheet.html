3-Command Setup

cd Run-Worldclass-computer-locally
./preflight.sh  &&  ./setup.sh        # Windows: .\preflight.ps1 ; .\setup.ps1

Daily Commands

ActionCommand
Start a chat ./chat.sh
Exit the chat /bye
List your models ollama list
Pull new model ollama pull <name>
Remove model ollama rm <name>
Restart (mac) brew services restart ollama
Restart (linux) sudo systemctl restart ollama
Uninstall ./uninstall.sh

The 5 Skills talk naturally to Claude Code

Say thisWhat happens
"Ask my local model …" Routes to local
"Summarize this file privately" Local summary, never uploaded
"Benchmark my model" Tokens/sec + TTFT
"Compare local vs cloud on X" Side-by-side answers
"Switch to <model>" Changes active model

Troubleshooting

SymptomFix
ollama: not found Re-run ./setup.sh
Chat hangs Restart service
model not found ollama pull <model>
RAM maxed / swapping Swap to a smaller model
First response slow Normal — model loading
Setup stalls on download Ctrl-C, re-run (resumes)
